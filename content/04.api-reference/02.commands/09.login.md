登录到您的远程缓存提供商。

```bash title="Terminal"
turbo login
```

默认提供商是 [Vercel](https://vercel.com/)。要指定不同的提供商，请使用 `--api` 选项。

## 标志选项

### --api \<url>

设置远程缓存提供商的 API URL。

```bash title="Terminal"
turbo login --api=https://acme.com/api
```

### --login \<url>

设置应动态生成令牌的登录请求的 URL。

```bash title="Terminal"
turbo login --login=https://acme.com
```

### --sso-team \<team>

通过提供您的团队 slug 连接到启用 SSO 的团队。

```bash title="Terminal"
turbo login --sso-team=slug-for-team
```

### --manual

手动输入令牌，而不是从登录服务请求令牌。
