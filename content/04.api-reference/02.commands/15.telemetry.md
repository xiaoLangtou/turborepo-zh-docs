---
title: telemetry
description: 管理此机器的遥测数据收集设置
---

```bash title="Terminal"
turbo telemetry [argument]
```

管理此机器的遥测数据收集。

<LinkToDocumentation href="/docs/telemetry">
  了解更多关于 Turborepo 遥测的信息。
</LinkToDocumentation>

## 参数

### `status`

获取此机器遥测的当前状态。

```bash title="Terminal"
turbo telemetry status
```

### `enable`

为此机器启用遥测。

```bash title="Terminal"
turbo telemetry enable
```

### `disable`

为此机器禁用遥测。

```bash title="Terminal"
turbo telemetry disable
```
